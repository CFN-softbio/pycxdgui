# Simple GUI for xray image viewing/analysis

See buildenv.sh for a sample script on how to setup. Requires anaconda.


Running:
    python startGUI.py

Dependencies:

- python3.5 PyQt4 numpy h5py pip
- pims pillow tqdm scipy scikit-image pandas cython
